Silabe, șva și dagheș

Silabe

O silabă este un grup de litere care se pronunță împreună într-un cuvânt. Cuvântul "silabă" are trei silabe: si-la-bă. Așa cum româna are reguli care reglementează silabele, la fel și ebraica. În ebraică:

➡ o silabă nu poate începe cu o vocală

➡ există două feluri de silabe:

➠ silabă deschisă, formată din consoană + vocală שָׁ ;
➠ silabă închisă formată din consoană + vocală + consoană לַח ;
שָׁ   לַח ➟ שָׁלַח
אָ   תוֹת ➟ אָתוֹת
כָּ   בֵד ➟ כָּבֵד

Șva

Cele trei vocale reduse prezentate, hatef qameț ֳ☐ , hatef segol ֱ☐ ; și hatef patach ֲ☐ ; sunt formate din vocalele respective și încă un semn ְ☐ care e numește șva (sau șeva).
Șva ְ☐ se găsește deseori singur sub consoane și are două sarcini.
➡ Șva vocal care se pronunță ca un ă scurt (fonetic ə) דְּבָרִים
➡ Șva mut care nu se promnunță dar are rol de a încheia silaba מִדְבָּר

➟ În cazul în care înaintea consoanei sub care se află șva se află o vocală scurtă, sheva este mut. מִדְבָּר , יַקְטִיל
➟ Dacă șva se află sub prima consoană a unui cuvânt, șva este vocal (nu este precedat de o vocală scurtă). דְּבָרִים
➟ Dacă două consoane consecutive au șva, primul este tăcut, iar al doilea este vocal. (Primul este precedat de o vocală scurtă și este mut, al doilea nu este precedat de vocală scurtă și este vocal). יִקְטְלוּ

Dagheș

Dagheș este și el de două feluri. Dagheș slab este punctul care schimbă pronunțarea literelor begadkefat, adică בּ ,כּ și פּ
Dagheș tare dublează o consoană, ceea ce înseamnă că sunt prezente două consoane, chiar dacă este scrisă doar una. Cuvântul קִטֵּל putea fi transliterat în limba română ca qittel (qit tel), deoarece dageș înseamnă că există două litere "t" ט. O consoană cu dagesh ar trebui pronunțată de două ori, ca în românescul "înnebuni".
Dacă dagheș se află într-o consoană alta decât begadkefat atunci el este dagheș tare. Dar consoanele begadkefat pot avea dagesh tare sau slab.

➟ Dacă sunetul care precede begadkefat este o consoană atunci dagheș este slab. מִדְבָּר
➟ Dacă sunetul care precede begadkefat este o vocală atunci dagheș este tare. קִדַּשׁ
ע , ח , ה , א și uneori ר sunt consoane guturale. Ele nu primesc niciodată dagheș.
